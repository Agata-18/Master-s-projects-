First task: Stationary vs ergodic

Let \(\epsilon_t \sim GW N(Z, \sigma^2)\), with \(GW N(Z, \sigma^2)\) denoting a Gaussian white noise process with:

  • Fixed variance \(\sigma^2\)

  • Random mean \(Z \sim U[-5, 5]\), where \(U\) represents the Uniform distribution.

  • \(Z\) is not indexed by \(t\), thus it does not change with \(t\), unlike \(\epsilon_t\).

  • The processes \(\{\epsilon_t\}_{t=-\infty}^{\infty}\) and \(Z\) are assumed to be independent. The process of interest is defined as: \[ X_t = \theta X_{t-1} + \epsilon_t \]

a) We first want to check whether and when the process is stationary. Using the definition of weak stationarity, prove that the process \(\{X_t\}_{t=-\infty}^{\infty}\) has constant mean, variance and autocovariance function when \(|\theta| < 1\). Intuitively, if the process is stationary, then the value of Z determines the equilibrium state of the process. Note: As the proof will involve two nested random quantities, you may need to use the Law of Total Expectations and the Law of Total Variance.

We know that the conditions for a process to be weak stationary are:

  1. \(\mathbb{E}(X_t) = \mu < \infty, \, \forall t\) (constant mean)
  2. \(\text{Var}(X_t) = \gamma_0(t) = \gamma_0 = \sigma^2 < \infty, \, \forall t\) (constant variance)
  3. \(\text{Cov}(X_t, X_{t+h}) = \gamma_h(t) = \gamma_h, \, \forall t, \forall h\) (covariance depends only on lag \(h\), not \(t\)) 1. MEAN We start by computing the mean to see whether the time series average is time invariant.

\[ \mathbb{E}(X_t) = \mathbb{E}(\theta X_{t-1} + \epsilon_t) \]

Using the linearity of expectation: \[ \mathbb{E}(X_t) = \theta \mathbb{E}(X_{t-1}) + \mathbb{E}(\epsilon_t) = \theta (X_{t-1}) + \mathbb{E}(Z) \]

At this point we have to consider that the \[\mathbb{E}(\epsilon_t) \] depends on the realizations of Z since it behaves as a Uniform random variable. So we have to condition the expectation on Z. In this case we can use the Law of total expectations: \[ \mathbb{E}(X) = \mathbb{E}[\mathbb{E}(X|Y)] \] So we proceed by using this law and by expanding the time series. This allows us to calculate the mean value of a time series up to each point. By doing so, we can follow how the mean of your time series changes with time:

\[ \mathbb{E}(X_t) = \mathbb{E}[\mathbb{E}(X_t|Z)]= \mathbb{E}[{E}(\theta X_{t-1} + \epsilon_t|Z)] = \mathbb{E}[\mathbb{E}(\theta (\theta X_{t-2} + \epsilon_{t-1}) + \epsilon_t))|Z] = \mathbb{E}[\mathbb{E}(\theta^2(\theta X_{t-3} + \epsilon_{t-2}) + \epsilon_{t-1} + \epsilon_t)|Z] \]

As we can see, if we continue to expand the series we have a succession of past errors till time t and we can represent it as a geometric summation. The reason behind this mathematical choice is that there is a succession of numbers such that the ratio of an element to its preceding one is always constant. So, the process becomes: \[ \mathbb{E}[{E}(\sum_{i=0}^{\infty}\theta^i X_{t-i} + \theta^i \epsilon_{t-i})|Z] \] The first term \[ \sum_{i=0}^\infty \theta^i X_{t-i} \] can be posed as equal to 0 when \(|\theta|<1\) beacause it converges geometrically and it wouldnt influence the time series. Specifically, when \(i\) increases the contribution of each \(X_{t-i}\) will diminish over time. Since \(\epsilon_t\) and \(\theta^i\) are independent, the second term \[ \mathbb{E}(\sum_{i=0}^{\infty}\theta^i \epsilon_t) \] can be rewritten as: \[ \mathbb{E}(\sum_{i=0}^{\infty}\theta^i){E}(\epsilon_{t-i}) \] Considering that the geometric series computed in the first expectation will converge to \(\frac{1}{1-\theta}\) when \(|\theta|<1\), we can simplify like this: \[ \frac{1}{1-\theta} \mathbb{E}(\epsilon_{t-i}) \] From the assumptions made in the assignments task \(\epsilon_t\) has mean equal to \(Z \sim U[-5, 5]\), so we can use the expected value of a generic uniform \[ \mathbb{E}(X) = \frac{a + b}{2}\] that in this case is: \[\mathbb{E}(Z) = \frac{-5 + 5}{2} = 0\] \[ \mathbb{E}(\frac{1}{1-\theta}Z) = \mathbb{E}(\frac{Z}{1-\theta}) = 0 \] So we can conclude that the process is centered around 0 and the mean is constat over time. Basically, we can make a first assumption of stationarity.

VARIANCE Now we can see if the second condition is met analysing the variance. Since we have to condition on Z, we can use the law of total variance. This latter states that the variance of a random variable X condioned on another random varible Y can be written like this:

\[ \text{Var}(Y) = \mathbb{E}[\text{Var}(Y | X)] + \text{Var}(\mathbb{E}[Y | X]) \]

Using this Law, we can write our variance like this:

\[ \text{Var}(X_t) = \mathbb{E}_Z[\text{Var}(X_t | Z = z)] + \text{Var}_Z[\mathbb{E}(X_t | Z = z)] \]

Proceeding with the calculations, we know that \[\mathbb{E}[Z]=0\] and \[\text{Var}(Z) = \frac{(b-a)^2}{12}= \frac{100}{12}\] :

\[ \text{Var}(X_t) = \mathbb{E}_Z[\text{Var}(X_t|Z=z)] + \frac{25}{3}[\frac{Z}{1-\theta}] \]

We can focus on the first term. \[ \mathbb{E}_Z[\text{Var}((\sum_{i=0}^{\infty}\theta^i X_{t-i} + \theta^i \epsilon_{t-i})|Z)] \] For the same reasons as before we can also not consider the first term.

\[ \mathbb{E}_Z[\text{Var}(\sum_{i=0}^{\infty}\theta^i X_{t-i}|Z)+\text{Var}(\theta^i \epsilon_{t-i})|Z))]= \mathbb{E}_Z[\frac{1}{1-\theta^2}\sigma^2] \] So the final result is:

\[ \text{Var}(X_t) = \mathbb{E}_Z[\frac{\sigma^2}{1-\theta^2}] + \frac{25}{3}[\frac{Z}{1-\theta}] = \frac{\sigma^2}{1-\theta^2} + \frac{25}{3}[\frac{Z}{1-\theta}] \]

The term \(1-\theta^2\) ensures that the variance is constant when \(|\theta| < 1\). As a last proof of the process stationarity we can compute the autocovariance.

AUTOCOVARIANCE

The autocovariance of a time series is given by: \[ \gamma_{t,t+h} = \mathbb{E}[(X_t - \mu_t)(X_{t+h} - \mu_{t+h})] = \gamma_h] \]

In our case we have that for \(t\): \[ X_t = \sum_{i=0}^{\infty}\theta^i X_{t-i} + \theta^i \epsilon_{t-i} = \sum_{i=0}^{\infty}\theta^i \epsilon_t \] and for \(t+h\) we get: \[ X_t = \theta^h X_t + \sum_{i=0}^{h-1} \theta^i \epsilon_{t+h-i} \] So we can rewrite the autocovariance substituting these definitions as follows: \[ \text{Cov}\left(X_t, \theta^h X_t + \sum_{i=0}^{h-1} \theta^i \epsilon_{t+h-i} \right) \] We want to express everything with respect to \(X_t\) and since it doesnt depend on the error term \(\epsilon_{t+h-i}\), we have that:

\[ \text{Cov}(X_t, X_{t+h}) = \text{Cov}(X_t, \theta^h X_t) = \theta^h \text{Cov}(X_t, X_t) = \theta^h \text{Var}(X_t) = \theta^h \gamma_0 \] and since \(\gamma_0\) is the variance, we can substitute the one we have computed beforehand: \[ = \theta^h \left( \frac{\sigma^2}{1 - \theta^2} + \frac{1}{(1 - \theta)^2} 10^{12} \right), \quad \forall |\theta| < 1 \] The term \(\theta^h\) leads to a decreasing autocovariance for increasing \(h\). The stationarity is ensured and confirmed also here for \(|\theta| < 1\).

b)To assess ergodicity, let us now focus on the autocorrelation function. Show that:

\[ \rho_h > 0, \, \forall h \]

This contrasts with the intuitive idea of an ergodic setting. Intuitively, while stationarity refers to the presence of an equilibrium state, ergodicity tells us whether the equilibrium is unique (given \(\theta\)).

Since we have proved that \[ \text{Cov}(X_t, X_{t+h}) = \theta^h \text{Var}(X_t) \] The autocorrelation function \(\rho_h\) is:

\[ \rho_h = \frac{\text{Cov}(X_t, X_{t+h})}{\text{Var}(X_t)} = \frac{\gamma_h}{\gamma_0} = \theta^h \]

Since \(Z\) is randomly drawn from a uniform distribution and does not vary over time, the time average of \(X_t\) would depend on the realization of \(Z\). Therefore, the process is not ergodic because the time averages would not converge to a single value across all realizations of \(Z\), implying non-ergodicity.

Autocorrelation Estimator \(\hat{\rho}_h\)

The estimator \(\hat{\rho}_h\) likely refers to the sample autocorrelation at lag \(h\), which is used to estimate the correlation structure between \(X_t\) and \(X_{t+h}\). The estimator for the autocorrelation at lag \(h\) is typically computed as:

\[ \hat{\rho}_h = \frac{\hat{\text{Cov}}(X_t, X_{t+h})}{\hat{\text{Var}}(X_t)} \]

where: - \(\hat{\text{Cov}}(X_t, X_{t+h})\) is the sample covariance between \(X_t\) and \(X_{t+h}\), - \(\hat{\text{Var}}(X_t)\) is the sample variance of \(X_t\).

Consistency of \(\hat{\rho}_h\)

For the estimator \(\hat{\rho}_h\) to be consistent, it must converge to the true autocorrelation \(\rho_h\) as the sample size \(T\) tends to infinity.

  1. For weakly stationary processes, the autocorrelation function \(\rho_h\) does not depend on time. In this case, the estimator \(\hat{\rho}_h\) is consistent for each fixed realization of \(Z\). However, since \(Z\) itself is random, the average autocorrelation across different realizations will not converge to a single value, violating ergodicity and causing non-consistency in the sample autocorrelation over multiple simulations.
  2. The process \(X_t\) depends on the random mean \(Z\), introducing variability in the autocorrelation structure across different realizations of \(Z\). For each realization of \(Z\), the process may be weakly stationary, but since \(Z\) is random, the autocorrelation will vary across simulations. Hence, \(\hat{\rho}_h\) will not converge to a single value across different realizations of \(Z\), violating consistency across simulations.

Conclusion

  • Stationarity: The process can be weakly stationary for a fixed value of \(Z\), but it is not stationary in the general case because \(Z\) is random.
  • Ergodicity: The process is not ergodic due to the random mean \(Z\), as different realizations of \(Z\) lead to different dynamics for \(X_t\).
  • Consistency of \(\hat{\rho}_h\): The estimator \(\hat{\rho}_h\) will be consistent for each fixed realization of \(Z\), but it will not be consistent over time or across different realizations of \(Z\), meaning it will not give the same autocorrelation across different simulations.
